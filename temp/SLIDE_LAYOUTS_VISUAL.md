## 🎨 Design Consistency Rules

### Spacing:
- Top margin: 0.5 inches
- Side margins: 0.5 inches
- Between elements: 0.3-0.5 inches

### Fonts:
- Titles: Segoe UI, 40pt, Bold
- Subtitles: Segoe UI, 20-24pt, Regular
- Body text: Segoe UI, 18-20pt, Regular
- Small text: Segoe UI, 14-16pt, Regular

### Cards:
- Background: White (#ffffff)
- Border: 1pt, Light Gray (#e2e8f0)
- Corner radius: 8-12pt
- Shadow: Subtle (optional)

### Bullets:
- Use ✓ (checkmark) instead of standard bullets
- Color: #10b981 (green) on light backgrounds
- Color: White on colored backgrounds

### Alignment:
- Titles: Left-aligned (except slides 1, 6, 8, 12 which are centered)
- Body text: Left-aligned
- Cards: Evenly distributed with equal spacing

---

## 💡 Quick Tips

1. **Use PowerPoint's Align tools**: Select multiple objects → Format → Align → Distribute Horizontally/Vertically

2. **Create one card, then duplicate**: Make one perfect card, then copy-paste and change content

3. **Use guides**: View → Guides to help align elements consistently

4. **Group objects**: Select multiple objects → Right-click → Group (makes moving easier)

5. **Save as template**: Once you have the design right, save as .potx template for future use

---

## ✅ Visual Checklist

For each slide, verify:
- [ ] Background color matches specification
- [ ] All text is readable (good contrast)
- [ ] Elements are aligned properly
- [ ] Spacing is consistent
- [ ] Fonts are Segoe UI
- [ ] Colors match the palette
- [ ] No spelling errors
- [ ] Emojis display correctly
